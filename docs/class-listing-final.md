- 기업별로 자소서를 **복붙**하고 있는 분

### 커리큘럼

공식을 설명하고 → AI가 바로 적용하는 걸 보여주는 라이브 구성입니다.

---

**[5분] OT — 왜 당신의 자소서는 5초 만에 넘겨지는가**

- 채용담당자의 심사 현실 (1건당 5초)
- "거의 모든 지원자가 같은 실수를 한다" — 4년간 60건+ 첨삭에서 발견한 패턴
- 오늘의 무기: "결이요" 공식 / 미끼 전략 / 키워드 체크리스트 / **그리고 이걸 자동으로 해주는 AI 도구**

---

**[15분] 1교시 — 탈락 패턴 진단 + AI가 자동으로 잡아내는 걸 봅니다**

7가지 탈락 패턴을 설명하고, **바로 AI에 자소서를 넣어서 자동 진단되는 걸 시연**합니다.

- ① **희망형 종결어미**: "~하고 싶습니다" 문서당 5~8회 → AI가 6건 자동 발견
- ② **감정·감상 과다**: "보람을 느꼈습니다" → AI가 전부 찾아서 삭제 대상 표시
- ③ **추상적 성과**: "좋은 결과" → AI가 **수치화 코칭** (350ms→15ms 등 제안)
- ④ 시간순 나열 ⑤ 기업 연구 부족 ⑥ 중복 서술 ⑦ 직무 무관 소재
- **🤖 라이브 시연**: 실제 자소서 → `/cover-letter` 실행 → **38/100점** 진단 결과가 뜨는 순간

---

**[15분] 2교시 — "결이요"로 재구성 + AI Before/After 실시간 생성**

공식을 설명하면서, **AI가 같은 자소서를 실시간으로 재구성하는 걸 보여드립니다.**

- **"결이요" 프레임워크**: 결론(5초) → 이유(수치) → 요청(비전)
- **3가지 어조 전환**: 희망→실행 / 감정→행동 / 과정→결과
- **🤖 라이브 시연**: 지원동기 Before/After
  - Before: "네이버는 대한민국을 대표하는... 성장하고 싶습니다"
  - After: "Redis 캐싱으로 **350ms→15ms 단축**한 경험이 있습니다... **즉시 기여하겠습니다**"
- AI가 **키워드 체크리스트**(7가지 소스)를 자동 생성하고, 반영률 14%→85% 가이드

---

**[10분] 3교시 — 미끼 전략 + AI가 면접 질문 25개를 뽑아주는 걸 봅니다**

- **미끼 전략**: 자소서에 면접관이 물어보고 싶어할 포인트를 배치 → 답변 미리 준비 → **면접의 60~70%를 내 답변으로 채우기**
- **"이미 팀원처럼" 전략**: 앱 리뷰, 업데이트 히스토리를 분석해서 팀의 고민을 공유
- **🤖 라이브 시연**: AI가 자소서에서 **미끼 포인트 5개 자동 추출** + 예상 질문 25개 생성
- 보너스: `/mock-interview`로 **AI 면접관과 1문 시뮬레이션** (질문→답변→4차원 평가→피드백)
